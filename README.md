# Portfólio Pessoal - Denis Oliveira Junior

Projeto desenvolvido para a disciplina de Fundamentos da Programação Web.

## Sobre o Projeto

Este projeto consiste em um portfólio pessoal online desenvolvido utilizando:

- HTML5
- CSS3
- JavaScript

O objetivo do site é apresentar informações pessoais, formação acadêmica, conhecimentos técnicos, projetos e formas de contato.

## Funcionalidades

- Navegação entre seções
- Layout responsivo para dispositivos móveis
- Formulário de contato com validação em JavaScript
- Links para GitHub e LinkedIn
- Organização visual moderna e simples
